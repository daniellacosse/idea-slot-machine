# Idea Slot Machine [![CircleCI](https://circleci.com/gh/daniellacosse/idea-slot-machine.svg?style=svg)](https://circleci.com/gh/daniellacosse/idea-slot-machine)

## commands (osx only)

### `yarn start`

starts the development server(s)

### `yarn build`

creates a production build in `.artifacts`

### `yarn checks`

check to see if jest, stylelint, and eslint pass
